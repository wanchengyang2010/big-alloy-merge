### 质量系统

| 特性 | 公式 | 说明 |
|------|------|------|
| 质量定义 | `mass = radius × 10` | 正比于半径（线性），避免小半径球过轻 |
| 位置修正 | `ratio = m_other / (m1+m2)` | 轻球位移更大 |
| 法向冲量 | `impulse = rel_vn × (1.0 + DAMPING) × m_other / (m1+m2)` | 质量反比缩放 |
| 切向冲量 | `min(abs(impulse)×FRICTION, rel_vt)` | 库伦钳制，不反转切向速度 |

## 碰撞解决流程（每帧）

```
1. dt = min(dt, 0.05)          # 帧时间上限
2. cooldown tick               # 冷却计时
3. spawn_protect tick           # 保护期计时
4. game over check              # 超线检测（重力之前）
5. gravity + pos integration    # 重力加速（真空，无空气阻力）+ 位置积分
6. spatial grid build           # 空间哈希网格
7. magnetic attraction          # 同等级磁吸（仅 full 模式）
8. wall clamp                   # 墙壁约束（刚性：速度归零）
9. collision resolution ×10     # 碰撞分离
   ├─ 同等级+可合并→跳过；同等级+冷却中→正常碰撞（防穿模）
   ├─ 法向冲量：质量缩放 + DAMPING
   ├─ 切向摩擦：库伦摩擦冲量（球间接触）
   ├─ 过度修正 ×1.2（确保完全分离）
   └─ wall clamp after each    # 每次迭代后墙约束
10. merge check                 # 合成检测
11. 睡眠机制                    # 静止0.3s→深度休眠，碰撞唤醒
12. cleanup dead items          # 移除已合成球
```

## v2.0.0.0 新增物理特性

### 旋转物理（v2.0.1.0 大幅修正）

球可绕中心旋转。碰撞切向摩擦力同时产生扭矩。v2.0.1.0 修正了旋转过度剧烈的问题。

| 参数 | 值 | 说明 |
|------|-----|------|
| `ROTATION_ENABLED` | True | 旋转物理开关 |
| `ANGULAR_DAMPING` | 0.85 | 角速度**每秒**保留系数（dt无关指数衰减） |
| `WALL_ROTATIONAL_FRICTION` | 0.40 | 壁面旋转摩擦（触壁时抑制现有旋转比例） |
| `WALL_TORQUE_COUPLING` | 0.08 | 线速度→扭矩耦合系数（v2.0.1.0 从旧1.0大幅降低） |
| 角速度归零阈值 | 0.005 rad/s | 低于此值清零 |

**角速度衰减（v2.0.1.0 dt无关公式）：**
```
ω *= damping ** dt       # damping=0.85 → 1s后保留85%, 5s后保留44%
```
旧版 `ω *= 0.98` 每帧与帧率相关且衰减过快，新版与帧率无关。

**碰撞扭矩公式（同v2.0.0.0）：**
```
cross = nx·ty - ny·tx              (±1, 切向×法向符号)
delta_ω₁ = f_impulse × ratio₁ × cross × 2 / r₁
delta_ω₂ = f_impulse × ratio₂ × cross × 2 / r₂
```
其中 `I = ½·m·r²`（圆盘转动惯量），扭矩 = 摩擦力 × 半径。

**壁摩擦扭矩（v2.0.1.0 修正）：**
- 触壁时**先**抑制现有旋转：`ω *= (1 - wall_rotational_friction)`
- **再**施加微弱线速度→扭矩耦合（仅当|v|>1.0）：`Δω = v × μ_w × 0.08 / r`
- 左壁：`Δω += vy × μ_w × tc / r`（下落产生逆时针扭矩）
- 右壁：`Δω -= vy × μ_w × tc / r`
- 底部：`Δω -= vx × μ_w × tc / r`
- 顶部同样抑制旋转但不产生新扭矩

### 壁摩擦

| 参数 | 值 | 说明 |
|------|-----|------|
| `FRICTION_WALL` | 0.3 | 容器壁动摩擦系数 |

球触壁时沿壁面切线方向减速：`v_tangential *= (1 - μ_w)`。
替代旧版地面固定系数 0.88。

### 每球独立参数

v2.0.0.0 支持逐 Tier 覆盖物理参数：

| 参数 | 默认来源 | 可逐Tier覆盖 |
|------|----------|-------------|
| mass | radius × 10 | ✓ `TierDef.mass` |
| friction | 模式 `friction_ball` | ✓ `TierDef.friction` |
| elasticity | 模式 `damping` | ✓ `TierDef.elasticity` |

### 模式参数系统

所有物理参数存储在 `modes.json`，由 ModeManager 统一管理。
`Game._apply_mode_physics()` 从 ModeDefinition 读取并覆写 `constants.*`。
窗口缩放时所有参数等比例缩放（`s(v) = v × scale`），游戏体验不变。
